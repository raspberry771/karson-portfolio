Karson Badge Generator

A simple, intuitive web app that allows developers to create custom-styled badges for events.

Live Demo

The Challenge & The Solution

The main challenge was generating the badge graphic dynamically on the client-side without a backend. I solved this by using SVG templates that are manipulated with JavaScript, ensuring fast performance and providing an instant preview for the user as they type and drag elements across a canvas. I also had to make sure it reads the Excel sheets headers correctly and create elements from it. Then the user inputs background image and logo image into a draggable canvas for custom adjustments.

Key Features

Real-time badge preview
Customizable colors and text
Easy-to-copy Markdown
Fully responsive design

Technical Stack

  • Frontend: HTML5, CSS3, JavaScript
  • Hosting: Netlify
